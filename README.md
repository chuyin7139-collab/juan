 ## Descripcion
 Los usuarios actuales necesitan una forma eficiente de organizar sus tareas diarias, pero las aplicaciones tradicionales de listas de tareas carecen de interactividad en tiempo real, obligando a recargas manuales de la página para ver cambios, lo que genera una experiencia lenta y poco intuitiva.
 
 ### Valor Agregado del Framework:
React permite una actualización inmediata de la interfaz (reactividad) gracias a su manejo de estado virtual. Cuando el usuario marca una tarea como completada, agrega una nueva o filtra resultados, los cambios se reflejan al instante sin recargar la página. Esto mejora la experiencia del usuario, aumentando la productividad y la satisfacción.

##  Requerimientos Funcionales (El "Qué hace")

1. RF1: Agregar nueva tarea con título y descripción.
2. RF2: Editar el contenido de una tarea existente.
3. RF3: Marcar/desmarcar tarea como completada (toggle).
4. RF4: Filtrar tareas por estado: Todas, Pendientes, Completadas.
5. RF5: Eliminar tarea con confirmación visual.
